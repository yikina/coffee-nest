## 基于react-pic_app后端服务

## 项目架构
├─ src
├─common //公共dto和统一返回msg格式
├─events
├─modules //接口

## Running the app
```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
